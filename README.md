# Job_Tracker
by Kelian SEBAICI

## Contexte
J'ai créé ce projet dans le but d'avoir un outil simple pour suivre mes candidatures. Etant un ingénieur récemment diplômé, je voulais trouver un moyen efficace de gérer mon processus de recherche d'emploi tout en continuant à apprendre et à me développer dans mon domaine.

## Fonctionnalités
- **Ajouter une candidature** : Permet d'ajouter une nouvelle candidature avec des détails tels que le titre du poste, l'entreprise, la date de candidature, etc.
- **Afficher les candidatures** : Affiche une liste de toutes les candidatures avec leurs détails.
- **Filtrer les candidatures** : Permet de filtrer les candidatures par statut (en cours, accepté, refusé) ou par entreprise.
- **Afficher des statistiques** : Affiche des statistiques sur les candidatures, comme le nombre total de candidatures, le taux de réponse, etc.
- **Exporter les données** : Permet d'exporter les données des candidatures au format CSV pour une utilisation ultérieure ou pour les partager.
- **Interface utilisateur simple** : Une interface épurée et facile à utiliser pour gérer les candidatures.

## Technologies utilisées
- **HTML/CSS** : Pour la structure et le style de l'application.
- **JavaScript** : Pour la logique de l'application, y compris la gestion des candidatures et l'exportation des données.
- **IndexedDB** : Pour stocker les données des candidatures localement dans le navigateur.
- **PWA (Progressive Web App)** : Pour permettre une expérience utilisateur fluide et la possibilité d'installer l'application sur différents appareils.

## Installation et utilisation
 Vous pouvez accéder à une version hébergée de l'application à l'adresse suivante : [https://s-kelian.github.io/job-tracker/](https://s-kelian.github.io/job-tracker/)

Si vous êtes sur Chrome ou Edge, vous pouvez installer l'application en cliquant sur l'icône d'installation dans la barre d'adresse ou en utilisant le menu du navigateur.
Vous pourrez dans ce cas accéder à l'application directement depuis votre bureau ou votre écran d'accueil, et elle fonctionnera même hors ligne grâce à sa nature de PWA.
